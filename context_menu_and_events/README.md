## Overview

This plugin shows how to work with editor events and extend the context menu using plugins.

It is called "Context menu and events" in the interface and isn't installed by default in cloud, [self-hosted](https://github.com/ONLYOFFICE/DocumentServer) and [desktop version](https://github.com/ONLYOFFICE/DesktopEditors) of ONLYOFFICE editors.

## How to use

1. Open the Plugins tab and press "Context menu and events".
2. Select text or an object in the document.
3. Open the context menu (right-click) to see additional menu items added by the plugin.
4. Click a custom menu item to trigger plugin logic based on the current editor state or selection.

If you need more information about how to use or write your own plugin, please see this https://api.onlyoffice.com/plugin/basic
